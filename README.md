# Phone-Directory-App
---

Phone Directory Application Developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add Contacts
- Edit Contacts
- List Contacts
- Delete Contacts
- Login Model
- Registration Model
- Responsive layout

## Technology stack
- Java 8+
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
- Tomcat server 8+
